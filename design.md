# Worktime App - Дизайн и UX

## Обзор

Приложение Worktime App предназначено для учета рабочего времени сотрудников на мобильных устройствах (iOS и Android). Интерфейс разработан с учетом принципов Apple Human Interface Guidelines и ориентирован на портретную ориентацию (9:16) с поддержкой одноручного управления.

## Экраны приложения

### 1. Главный экран (Home Screen)

**Назначение:** Центральный элемент приложения для быстрого доступа к учету рабочего времени.

**Основные элементы:**
- **Статус пользователя** (верхняя часть): Крупный текст, показывающий текущий статус (например, "На работе", "На перерыве", "Рабочий день завершен")
- **Таймер** (центр): Отображение текущего времени на работе или до завершения перерыва
- **Кнопки действий** (основная часть): Четыре больших кнопки для основных операций:
  - "Начать работу" / "Завершить работу" (красная/зеленая)
  - "Начать перерыв" / "Завершить перерыв" (желтая)
  - "Выйти с рабочего места" / "Вернуться" (серая)
  - "Задачи" (синяя, ведет на экран задач)
- **Информация о дне** (нижняя часть): Краткая сводка отработанного времени за день
- **Нижняя панель навигации**: Пять вкладок (Home, Tasks, Calendar, Reports, Settings)

### 2. Экран задач (Tasks Screen)

**Назначение:** Управление задачами и отслеживание времени по задачам.

**Основные элементы:**
- **Кнопка создания задачи** (верхняя часть): Плавающая кнопка "+" для добавления новой задачи
- **Список задач** (основная часть): Список активных задач с отображением:
  - Названия задачи
  - Времени, затраченного на задачу
  - Статуса задачи (активная, завершена, пауза)
  - Кнопки управления (запуск/остановка таймера)
- **Фильтры** (верхняя часть): Вкладки для фильтрации (Все, Активные, Завершенные)

### 3. Экран календаря (Calendar Screen)

**Назначение:** Визуальное представление рабочего времени и статусов дней.

**Основные элементы:**
- **Месячный календарь** (основная часть): Сетка с днями месяца, каждый день имеет цветовую индикацию:
  - Зеленый: Рабочий день с нормальным временем
  - Желтый: Рабочий день с аномалией
  - Серый: Выходной/праздник
  - Синий: Отпуск/больничный
- **Информация о дне** (при нажатии): Модальное окно с деталями дня (начало/конец работы, перерывы, задачи)
- **Навигация по месяцам** (верхняя часть): Стрелки для перемещения по месяцам

### 4. Экран отчетов (Reports Screen)

**Назначение:** Просмотр аналитической информации о рабочем времени.

**Основные элементы:**
- **Выбор типа отчета** (верхняя часть): Сегментированный контроль для выбора (День, Неделя, Месяц, Год)
- **Визуализации** (основная часть): Графики и диаграммы, показывающие:
  - Распределение времени по дням
  - Распределение времени по задачам
  - Тренды рабочего времени
- **Табличное представление** (нижняя часть): Детальная таблица с данными отчета

### 5. Экран настроек (Settings Screen)

**Назначение:** Персонализация приложения и управление учетной записью.

**Основные элементы:**
- **Профиль пользователя** (верхняя часть): Информация о пользователе (имя, роль)
- **Настройки приложения** (основная часть):
  - Язык интерфейса (Русский, Английский)
  - Тема оформления (Светлая, Темная, Автоматически)
  - Уведомления (включение/отключение)
  - Информация о приватности GPS
- **Действия** (нижняя часть):
  - Смена пароля
  - Выход из аккаунта

## Интерактивные элементы

### Кнопки

**Первичная кнопка** (основные действия):
- Фон: Акцент (Primary)
- Текст: Белый
- Высота: 48px
- Скругление: 12px
- Состояние нажатия: Масштаб 0.97, снижение прозрачности

**Вторичная кнопка** (дополнительные действия):
- Фон: Поверхность
- Текст: Основной текст
- Граница: 1px, цвет границы
- Высота: 48px
- Скругление: 12px

### Карточки

- Фон: Поверхность
- Граница: 1px, цвет границы
- Скругление: 12px
- Отступ: 16px
- Тень: Легкая тень (iOS style)

## Анимации

- **Переходы между экранами**: 300ms, плавное затухание
- **Нажатие кнопки**: 80ms, масштабирование 0.97
- **Появление элементов**: 250ms, затухание
- **Обновление данных**: Плавное обновление без резких скачков

## Адаптивность

- **Портретная ориентация**: Основной режим работы
- **Ландшафтная ориентация**: Поддерживается, элементы адаптируются
- **Безопасные области**: Учитываются вырезы (notch) на iPhone X+ и домашняя кнопка на iPhone
- **Размеры экранов**: Поддержка от 4.7" до 6.7" (стандартные размеры мобильных устройств)

## Доступность

- **Контраст текста**: Минимум WCAG AA (4.5:1 для основного текста)
- **Размер сенсорных целей**: Минимум 44x44pt
- **Поддержка VoiceOver/TalkBack**: Все интерактивные элементы имеют описания
- **Поддержка системных настроек**: Приложение реагирует на настройки доступности устройства
